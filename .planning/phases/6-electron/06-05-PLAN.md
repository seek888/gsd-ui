# Plan 06-05: 构建和测试

## 目标
配置 Electron Builder 并生成可分发的应用包。

## Tasks

### Task 1: 配置 electron-builder
创建 `electron-builder.json`:
```json
{
  "appId": "com.gsd.ui",
  "productName": "GSD UI",
  "directories": {
    "output": "release"
  },
  "files": [
    "dist/**/*",
    "dist-electron/**/*"
  ],
  "mac": {
    "target": ["dmg", "zip"],
    "category": "public.app-category.developer-tools"
  },
  "win": {
    "target": ["nsis"],
    "icon": "public/icon.ico"
  },
  "linux": {
    "target": ["AppImage"],
    "category": "Development"
  },
  "nsis": {
    "oneClick": false,
    "allowToChangeInstallationDirectory": true
  }
}
```

### Task 2: 更新 package.json scripts
```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview",
    "electron:dev": "vite",
    "electron:build": "npm run build && electron-builder",
    "electron:build:mac": "npm run build && electron-builder --mac",
    "electron:build:win": "npm run build && electron-builder --win",
    "electron:build:linux": "npm run build && electron-builder --linux"
  }
}
```

### Task 3: 生成应用图标
```bash
# 创建 1024x1024 图标
mkdir -p public
# 添加图标文件到 public/icon.png
```

### Task 4: 执行构建
```bash
# macOS
npm run electron:build:mac

# Windows (在 Windows 上执行)
npm run electron:build:win
```

### Task 5: 验证构建产物
```bash
# macOS
ls -la release/mac/
# 应该包含: GSD UI.dmg, GSD UI.app

# Windows
ls -la release/win/
# 应该包含: GSD UI Setup.exe
```

## 验证清单
- [ ] macOS .app 可以启动
- [ ] macOS .dmg 可以安装
- [ ] Windows .exe 可以安装
- [ ] 核心功能正常工作（CLI 检测、文件浏览、命令执行）

## 注意事项
- Electron 包大小约 150MB，比 Tauri (~10MB) 大很多
- 这是 Electron 的已知限制

## 迁移完成标准
1. 应用可以在 macOS 和 Windows 上构建
2. 应用可以正常启动并显示 UI
3. 核心功能（CLI 检测、命令执行、文件浏览）正常工作
