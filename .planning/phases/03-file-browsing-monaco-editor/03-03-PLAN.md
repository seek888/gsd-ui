    - wordWrap: 'on'
    - fontSize: 14
    - lineHeight: 22
    - tabSize: 2
    - automaticLayout: true (handles resize)

    onMount callback:
    - Store editor ref in useState (for layout calls)
    - Add keyboard shortcut: editor.addCommand(monaco.KeyMod.CtrlCmd | monaco.KeyCode.KeyS, () => { onSave(); })

    Resize handling:
    - Use useEffect with ResizeObserver on parent container
    - Call editor.layout() when container size changes
    - This prevents common Monaco resize issue where content gets clipped

    Styling wrapper:
    - Wrap Editor in div with className="h-full"
    - Ensure parent has defined height for 100% to work
