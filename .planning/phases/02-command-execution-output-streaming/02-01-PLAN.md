| T-02-04 | Information Disclosure | Error messages | accept | Errors shown to user who triggered command |
| T-02-05 | Denial of Service | Command spam | accept | User can only spam their own local CLI |
| T-02-06 | Elevation | Privilege escalation | accept | claude CLI runs at user privileges, no elevation |
